# Documentation Images

This directory contains screenshots and images for the user manual.

## Required Images

Please add the following screenshots to this directory:

1. **dashboard-overview.png** - Full dashboard view showing all metrics and charts
2. **dashboard-metrics.png** - Close-up of the four metric cards
3. **simulator-form.png** - Product details input form
4. **simulator-results.png** - Prediction results card

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 1200-1600px width for screenshots
- **Naming**: Use lowercase with hyphens (e.g., `dashboard-overview.png`)
- **Quality**: Ensure text is readable and UI elements are clear

## Adding Images

1. Take screenshots of your running application
2. Save them in this directory with the names listed above
3. The USER_MANUAL.md will automatically reference them
